# Test PGMQ

> [!NOTE]
> You need to install [PGMQ](https://github.com/pgmq/pgmq/blob/main/INSTALLATION.md) in your Postgres DB.

- [SQLx CLI](https://github.com/launchbadge/sqlx/tree/main/sqlx-cli) (v0.8.6)
- [Grow-RS](https://github.com/Wilovy09/Grow-rs) (v2.2.0)
- [Just](https://github.com/casey/just) (v1.46.0)
- [Cargo Watch](https://crates.io/crates/cargo-watch) (v8.5.3)
- [MailHog](https://github.com/mailhog/MailHog) (v1.0.1)
- Postgres DB (v17.7)
